# Casper's HeadFeedr 🚀

一个极简、沉浸式的创意写作与头脑风暴工具。专为小说家、编剧和创作者设计。

👉 **[点击这里直接使用 App](https://padmaking.github.io/HeadFeedr/)** 👈

## ✨ 核心功能

* **📚 多项目管理**：支持创建多本小说，独立管理设定。
* **🤖 AI 灵感助手**：内置 DeepSeek/Gemini/OpenAI 接口，根据当前文风自动续写、生成 Logline 或大纲。
* **💬 沉浸式对话**：在写作界面直接与 AI 探讨剧情，上下文自动关联。
* **🔒 隐私优先**：所有数据（大纲、正文、API Key）仅保存在你浏览器的本地数据库（IndexedDB）中，不上传任何服务器。
* **📱 全平台适配**：手机、平板、电脑浏览器均可完美运行，支持 PWA 安装到桌面。
* **📦 导入导出**：支持一键备份整个项目库，数据安全无忧。

## 🛠️ 如何使用

1.  点击上方的链接打开网页。
2.  (可选) 在手机浏览器点击“分享” -> “添加到主屏幕”，获得原生 App 体验。
3.  在设置中填入你的 API Key (支持 Google Gemini, DeepSeek 等) 以开启 AI 功能。
4.  开始创作！
