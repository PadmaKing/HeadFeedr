# Casper's HeadFeedr 🚀

A minimalist, immersive creative writing and brainstorming tool. Designed for novelists, screenwriters, and creators.

👉 **[Click here to launch the App](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)** 👈

## ✨ Key Features

* **📚 Multi-Project Management**: Create and manage multiple novels or stories, each with its own independent settings and database.
* **🤖 AI Inspiration Assistant**: Built-in integration for DeepSeek, Google Gemini, and OpenAI. Brainstorm loglines, generate plot outlines, or get writing suggestions based on your specific style.
* **💬 Immersive Chat**: Discuss your plot and characters directly with the AI co-writer within the editor interface. It reads your current draft context automatically.
* **🔒 Privacy First**: All data (outlines, drafts, characters, and API Keys) is stored **only** in your browser's local database (IndexedDB). Nothing is uploaded to any external server.
* **📱 Cross-Platform**: Runs perfectly on mobile, tablet, and desktop browsers. Supports PWA (Progressive Web App) to install directly to your home screen.
* **📦 Import & Export**: One-click full backup and restore (JSON format) to ensure your data is always safe.

## 🛠️ How to Use

1.  Click the link above to open the web app in your browser.
2.  **(Optional)** On mobile (iOS/Android), tap the "Share" or "Menu" button and select **"Add to Home Screen"** for a native app-like experience.
3.  Go to **Settings** and enter your API Key (e.g., Google Gemini, DeepSeek) to unlock AI features.
4.  Start creating!

## ❤️ About

This is a lightweight writing tool I developed to remove distractions and focus purely on the story itself.
If you have any suggestions, feel free to open an Issue.



# Casper's HeadFeedr 🚀

一个极简、沉浸式的创意写作与头脑风暴工具。专为小说家、编剧和创作者设计。

👉 **[点击这里直接使用 App](https://padmaking.github.io/HeadFeedr/)** 👈

## ✨ 核心功能

* **📚 多项目管理**：支持创建多本小说，独立管理设定。
* **🤖 AI 灵感助手**：内置 DeepSeek/Gemini/OpenAI 接口，根据当前文风自动续写、生成 Logline 或大纲。
* **💬 沉浸式对话**：在写作界面直接与 AI 探讨剧情，上下文自动关联。
* **🔒 隐私优先**：所有数据（大纲、正文、API Key）仅保存在你浏览器的本地数据库（IndexedDB）中，不上传任何服务器。
* **📱 全平台适配**：手机、平板、电脑浏览器均可完美运行，支持 PWA 安装到桌面。
* **📦 导入导出**：支持一键备份整个项目库，数据安全无忧。

## 🛠️ 如何使用

1.  点击上方的链接打开网页。
2.  (可选) 在手机浏览器点击“分享” -> “添加到主屏幕”，获得原生 App 体验。
3.  在设置中填入你的 API Key (支持 Google Gemini, DeepSeek 等) 以开启 AI 功能。
4.  开始创作！
